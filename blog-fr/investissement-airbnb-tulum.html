Tulum est l’un des premiers marchés Airbnb d’Amérique latine. Les chiffres réels, configuration par configuration.

Projection de revenus (villa 2 chambres — le sweet spot)

  • Haute saison (déc–avril) : 250–350 USD/nuit, 80–90 % d'occupation
  • Intersaison (mai–juin, nov) : 150–200 USD/nuit, 60–70 %
  • Basse saison (juil–oct) : 120–180 USD/nuit, 45–60 %

Revenu brut annuel estimé : 45 000–75 000 USD. Voir le calcul ROI complet avec charges.

Ce qui fait la différence

  • Piscine privée (indispensable : +30–40 % de revenus)
  • Rooftop avec plunge pool (+40–80 USD/nuit)
  • Design jungle/bohème photogénique
  • Photos professionnelles (+40 % de réservations)
